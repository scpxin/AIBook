# 参考项目记录文档

> 本文档记录用户分享的参考项目，以及从每个项目中借鉴并应用到本项目的具体内容。

## 格式说明

每个参考项目记录包含：
- **项目信息**：名称、URL、简介
- **借鉴内容**：从该项目学到了什么
- **应用位置**：在本项目中的哪些文件/模块落地了这些改进

---

## 模板

### [项目名称](链接)

**项目简介**: 一句话说明项目是什么

**借鉴内容**:
1. [借鉴点1] → 应用位置：`文件路径:行号`
2. [借鉴点2] → 应用位置：`文件路径:行号`
